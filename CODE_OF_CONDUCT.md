# Code of Conduct

This project follows the **Contributor Covenant, version 2.1**.
The full text is canonical at:

https://www.contributor-covenant.org/version/2/1/code_of_conduct/

It applies to all project spaces — issues, pull requests, code
review, and any other forum where contributors interact under the
project's banner.

## Reporting

Concerns or violations can be reported privately to the maintainer
listed in [`SECURITY.md`](SECURITY.md). Reports are handled
confidentially. Reporters can expect an acknowledgement within a
few business days and a follow-up once the maintainer has reviewed.
